# #🌊 Sea-Level Rise Impact Dashboard

# 📊 Project Overview
This project presents an interactive Streamlit dashboard designed to analyse the global impact of sea-level rise across multiple indicators and scenarios. The dashboard enables users to explore how rising sea levels affect land, population, and economic activity across different countries and regions.

The dataset includes country-level data for six key indicators:
- Land
- Population
- GDP
- Agriculture
- Wetland
- Urban Extent

All indicators are analysed across five sea-level rise scenarios ranging from **1 meter to 5 meters**.

---

# 🎯 Objectives
- Visualise global exposure trends across sea-level rise scenarios  
- Compare impacts across different indicators  
- Identify the most vulnerable countries and regions  
- Enable interactive exploration using filters  

---

# ⚙️ Features
- 📌 KPI cards (Land, Population, GDP exposure)
- 📈 Global trend analysis across scenarios
- 🌍 Regional comparison by continent
- 🏆 Country-level vulnerability rankings
- 🔥 Heatmap of top 15 most exposed countries
- 📊 Combined Population & GDP growth chart
- 🎛️ Interactive sidebar filters (Scenario & Indicator)

---

# 🛠️ Technologies Used
- **Python**
- **Streamlit**
- **Pandas**
- **Plotly**

---

# 📂 Dataset
The dataset contains country-level data with the following columns:
- Country  
- Continent  
- Indicator  
- Scenario  
- Total  
- Impact  
- Percentage
